# 隐形水印小工具

这是一个使用 Rust 和 egui 开发的极简隐形水印工具，支持在图片中嵌入和提取盲水印，是经典的 [DWT+DCT+SVD 算法](https://github.com/guofei9987/blind_watermark/)的 [Rust 实现](https://github.com/naganohara-yoshino/blind-watermark-rust)的再封装，适合用于防盗图和维权提取场景。
